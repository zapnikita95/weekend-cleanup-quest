# Weekend Cleanup Quest

Пиксельная браузерная мини-игра для уборки по выходным вдвоём или для любой пары друзей.

## Что есть

- Простые профили по уникальной почте без подтверждений и паролей.
- Два активных игрока выбираются из сохранённых профилей или создаются прямо перед игрой.
- CSS-пиксельные аватарки и загрузка своих картинок.
- Общая локальная библиотека дел: название, примерное время, сложность, включение в текущий раунд.
- Рандомное распределение выбранных дел так, чтобы сумма времени на каждого игрока не превышала лимит раунда.
- Экран уборки: Space закрывает следующее дело первого игрока, Enter закрывает следующее дело второго.
- Клик мышкой по делу тоже отмечает его выполненным.
- Очки за выполненные дела, минуты, сложность, скорость, серию выполнений и оценку партнёра.
- История игр текущей пары, счёт побед внутри пары и общий лидерборд пар.
- Простая chiptune-музыка через Web Audio API, без внешних аудиофайлов.

## Хранение данных

Backend пишет данные в volume:

- база: `/data/weekend-cleanup-quest.json`
- загруженные аватарки: `/data/uploads`

Для локальной разработки можно переопределить путь:

```bash
DATA_DIR=./data npm start
```

На Windows PowerShell:

```powershell
$env:DATA_DIR="./data"; npm start
```

## Локально

```bash
npm install
npm run dev
```

Для проверки production-сервера:

```bash
npm run build
npm start
```

## Railway

Создай volume, смонтированный в `/data`, затем используй:

- Build command: `npm run build`
- Start command: `npm start`

Приложение само создаст JSON-базу и папку для аватарок при первом запуске.
