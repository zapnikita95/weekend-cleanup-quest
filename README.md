# Tidy Titans

**Играй в уборку — и дом становится чище.**

[Tidy Titans](https://tidytitans.ru) — пиксельная браузерная игра, которая превращает домашние дела в квест с очками, QR-ссылками и призами. Работает на компьютере и телефоне без установки.

## Режимы

| Режим | Для кого | Как работает |
| --- | --- | --- |
| **Одиночный** | Один человек | Отмечаешь дела, набираешь очки, получаешь приз по цели. Без самооценки. |
| **Парная игра** | Два и больше участников | Дела распределяются по времени. Участники оценивают друг друга после завершения. |
| **Квест для ребёнка** | Родитель + ребёнок | Ребёнок играет по ссылке/QR. Призы за бронзу, серебро и золото. |

## Как начать за 2 минуты

1. Открой [tidytitans.ru](https://tidytitans.ru)
2. Выбери режим и собери список дел
3. Задай приз или цель по очкам
4. Нажми «Сгенерировать уборку»
5. Открой QR на телефоне и отмечай дела
6. Нажми **«Завершить»** — откроются итоги с очками и призом

## Что умеет игра (V2 — большой апдейт)

- Пиксельные аватарки и свои картинки
- **Библиотека + шаблоны одним кликом**: Выходные, Ежедневный reset, Подросток, Глубокая уборка и др.
- **Teen Mode** — полноценный опыт для подростков: другие дела, награды про автономию, эквивалент денег, цели накопления
- QR + личный **кабинет ребёнка/подростка** с балансом звёзд, уровнем, стрики, магазином наград и целями
- Накопительный банк звёзд + долгосрочные цели (сбережения)
- Фото-подтверждения, рейтинг, история
- Достижения, лидерборды, статистика
- Chiptune + zero-install

Tidy Titans теперь не просто разовая игра, а система мотивации и роста самостоятельности.

## FAQ

**Нужна ли регистрация?**  
Достаточно общей почты семьи — без паролей и подтверждений.

**Можно ли не отметить все дела?**  
Да. Кнопка «Завершить» не отмечает невыполненные дела автоматически.

**Кто ставит баллы 0–3?**  
Только другой равноправный участник или родитель — не сам исполнитель.

**Где хранятся данные?**  
На сервере в `/data` (JSON + загруженные аватарки).

## Develop locally

```bash
npm install
npm run dev
```

Production build:

```bash
npm run build
npm start
```

Data directory override:

```powershell
$env:DATA_DIR="./data"; npm start
```

## Deploy (Railway)

- Build: `npm run build`
- Start: `npm start`
- Mount volume at `/data`
