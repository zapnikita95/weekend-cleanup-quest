# Tidy Titans — V2 Leap Plan (Weekend Cleanup Quest)

**Цель:** Превратить проект в лучший в классе издательский инструмент "задания от родителей детям / семье" с сильным геймификацией. К концу дня — ощутимо прокачанная версия, которая выглядит как зрелый продукт. Фокус на **ценности для пользователей**, удержании и расширении аудитории (включая подростков), без полной перерисовки фронтенда.

## Текущий статус (анализ)
- Сильные стороны:
  - Пиксель-арт + саунд (chiptune) — уникальный вайб, мгновенный fun.
  - 0-install браузер + QR flow (мгновенный старт на телефоне).
  - 3 режима: solo / duo / childQuest.
  - Уже есть ChildProfile + starBalance + ledger + redeem + achievements (категории).
  - Photo proof поддержка (частично).
  - История, лидерборды, синхронизация.
- Слабые места (по сравнению с топами: S'moresUp, OurHome, BusyKid, PointUp, Habitica, Homey, Greenlight):
  - Нет recurring / шаблонов задач (самое частое требование).
  - Слабая долгосрочная прогрессия (levels, streaks, goals).
  - Нет dedicated teen-опыта (подростки быстро отваливаются от "детского" гейма).
  - Магазин наград примитивный (нет категорий, нет "реальных" эквивалентов).
  - Аналитика поверхностная (нет трендов, insights для родителей).
  - Нет "целей накопления" (big goal / savings jar).
  - Данные в основном сессионные; профили только в childQuest.
  - Мало вирусности / семейной ценности (шаблоны, экспорт, family overview).

## Принципы развития (чтобы прыгнуть выше головы)
- **Не перерисовывать фронт ради фронта.** Только когда фича приносит новых пользователей или критично улучшает retention (teen mode, shop, goals).
- **Держать zero-friction:** QR, браузер, быстро.
- **Долгосрочный retention > one-off game:** Профили + банк + цели + streaks + levels.
- **Teen bridge:** Отдельный опыт для 12-17 без ломки kid flow. Подростки хотят автономию, деньги/привилегии, уважение.
- **Смотреть на победителей:**
  - S'moresUp: photo proof + AI + allowance + team challenges + badges.
  - PointUp: RPG levels, 40+ badges, streaks, team quests, photo.
  - BusyKid / Homey / Greenlight: allowance, financial literacy, payout, teen fit.
  - OurHome: approval, schedule, points to real incentives.
  - Habitica: deep RPG quests.
- **Ключевые дифференциаторы Tidy Titans V2:** Pixel soul + мгновенный доступ + мощный child/teen кабинет + реальные семейные механики + простота.

## Приоритетный роадмап (реалистично на день + дальше)

### Phase 1 (Сегодня — Big Leap)
1. **Task Templates & Quickstarts** (высокий impact)
   - Кнопки в setup: "Выходные стандарт", "Ежедневный 15-мин reset", "Подростковая самостоятельность", "Глубокая уборка", "Минимальный квест".
   - Загружают преднастроенные списки + секции.
   - Легко расширять.

2. **Teen Mode / Teen Profiles** (стратегически важно)
   - В child profile добавить ageGroup: 'kid' | 'teen'.
   - При teen: другие дефолтные задачи (ответственность, авто, покупки, стирка полный цикл и т.д.).
   - Другие формулировки наград: "Час без вопросов по телефону", "Выбор фильма всей семьей", "Эквивалент 200₽ на карман", "Поздний отбой".
   - Другой тон в кабинете ("Уровень самостоятельности", "Очки ответственности").
   - Отдельный набор достижений или те же с другим лейблингом.
   - В setup для childQuest — опция "Режим для подростка".

3. **Level + Streak система**
   - Level = 1 + floor(totalQuests / 3) или на базе starBalance.
   - Показывать в кабинете, в саммари, в истории.
   - Streak: последовательные дни/квесты (простой подсчет по датам в ledger).
   - Визуал: "Lv. X • 🔥 4 дня подряд".

4. **Улучшенный Reward Shop & Redemption**
   - Добавить category: 'privilege' | 'experience' | 'allowance' | 'item'.
   - В редакторе наград — выбор категории + описание.
   - Предзаполненные наборы для kid и teen.
   - В кабинете — группировка по категориям.
   - История выкупов (отдельный список + "Всего выкуплено").
   - Опционально: "Эквивалент в деньгах" (родитель задаёт курс "1 звезда = X ₽").
   - Родитель видит все redemption'ы детей.

5. **Long-term Goals (Сбережения)**
   - В профиле ребёнка/подростка: "Цель" (label + starsTarget).
   - Прогресс бар в кабинете: "Накоплено X / Y звёзд на [цель]".
   - Можно менять цель.
   - При redeem показывать влияние на цель.

6. **Родительский обзор + Insights (минимум)**
   - В stats / home — секция "Дети и профили" с быстрым просмотром баланса, уровня, последней активности нескольких профилей.
   - Простая статистика: % completion, total stars за период.
   - Кнопка "Экспорт отчёта" (CSV или копировать текст).

### Phase 2 (Следующие шаги, 1-3 дня)
- Полноценные recurring chores (с датами, авто-генерация следующего раунда).
- Web Push / простые напоминания (или "Скопировать текст напоминания").
- Photo approval flow (родитель подтверждает фото перед начислением).
- Skill tree / mini progression (уборка кухни прокачивает "Кухонный мастер" — бонусы).
- Sibling challenges / team mode.
- Allowance payout simulation (лог "выплачено").
- PWA + offline + install prompt.
- Лучшая вирусность: shareable "challenge links", красивые сертификаты.
- Расширенная библиотека задач + умные предложения.
- Интеграции (простые: Google Calendar export, или просто iCal).
- A/B копирайтинг и онбординг для родителей и подростков.

## Технические заметки
- Все изменения обратно совместимы (дефолты, опциональные поля).
- Основная логика в `src/App.tsx` + `childProgress.ts` + `server.mjs`.
- Данные: JSON в data/, uploads base64 или файлы.
- Избегать тяжёлых рефакторингов монолита App.tsx — добавлять функции + conditional UI.
- Для фото: уже dataURL, следить за размером.
- Новые поля в ChildProfile: `ageGroup`, `level` (computed), `currentGoal`, `streakDays`, `moneyRate` (курс).
- Шаблоны хранить константами в коде (легко править).
- Для teen: отдельные default teen chores + reward presets.

## Метрики успеха V2
- Родитель может за 30 сек задать шаблон + ребёнка/подростка + цель.
- Ребёнок/подросток видит в кабинете: баланс + уровень + цель + что можно выкупить прямо сейчас + достижения.
- Есть ощущение прогресса недели/месяца, а не только одной уборки.
- Подросток не чувствует себя "малышом".

## Маркетинг / позиционирование после прокачки
"Пиксельная игра, которая превращает уборку в настоящую систему ответственности. Для детей и подростков. Мгновенный старт по QR. Накопительные звёзды, цели, призы и рост самостоятельности."
